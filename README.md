MvvmKit
==========
[![Build Status](https://dev.azure.com/areaengineering/MvvmKit/_apis/build/status/MvvmKit?branchName=main)](https://dev.azure.com/areaengineering/MvvmKit/_build/latest?definitionId=37&branchName=main)
[![Build Status](https://dev.azure.com/areaengineering/MvvmKit/_apis/build/status/MvvmKit?branchName=development)](https://dev.azure.com/areaengineering/MvvmKit/_build/latest?definitionId=37&branchName=development)

MvvmKit is a cross-platform MVVM framework. It enables developers to create apps using the MVVM pattern on *dotnet-ios, dotnet-macos*. This allows for better code sharing by allowing you to share behavior and business logic between platforms.
